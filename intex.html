✅ 1. What is !DOCTYPE html> and why is it used?
Answer: It tells the browser that the document is using HTML5 standard.
It helps the browser render the page correctly.
✅ 2. What is the purpose of the html> tag?
Answer: It wraps the entire HTML document.
Everything (head + body) goes inside this.
✅ 3. What does lang="en" mean in the html> tag?
Answer: It sets the language of the document to English.
Helpful for SEO, translation tools, and screen readers.
✅ 4. What is the head> section used for?
Answer: It contains metadata about the page — like title, character set, CSS links, and scripts
It is not visible to users on the page
✅ 5. What is the role of meta charset="UTF-8" ?
Answer: It tells the browser to use UTF-8 character encoding, which supports all characters (English, Hindi, symbols, etc).
✅ 6. What is the purpose of meta name="viewport"...>?
Answer: It helps the website be responsive — adjusts layout on mobile/tablet/desktops properly.
It's essential for mobile-friendly websites.
✅ 7. What does the title tag do?
Answer: Sets the tab name (top of the browser window).
Also improves SEO.
✅ 8. What is the body> tag for?
Answer: It contains everything the user can see on the screen, like text, images, buttons, etc.
✅ 9. What is the difference between head> and body>?
head> - Contains page information , Hidden from user
body - Contains visible content , Shown to user on screen
✅ 10. Can a webpage work without DOCTYPE html>?
Answer: Yes, it can load — but it may not render correctly on all browsers.
You should always use it to ensure standard behavior.

⭐ types of tags
1) inline tag , container tag
2) block tag

1) Inline Tags

1 img – Image Tag
Web page pe image dikhane ke liye use hota
2 a – Anchor Tag Hyperlink banane ke liye use hota hai. Kisi page ya website pe le jaata hai
3 b – Bold Tag
Text ko bold (gadha) dikhata hai, sirf styling ke liye.
4 strong – Strong Tag
Text ko bold karta hai aur uski importance bhi show karta hai.
5 i – Italic Tag
Text ko italic (tedha) dikhata hai, sirf style ke liye.
6 em – Emphasis Tag
Text italic hota hai aur uspe focus ya zor dena dikhata hai.
6 u – Underline Tag
Text ke neeche line lagata hai (underline karta hai).
7 mark – Mark Tag
Text ko highlight karta hai, jaise yellow marker se mark kiya ho.
8 small – Small Tag
Text ko chhota (small size) dikhata hai.
9 sub – Subscript
Text ko neeche shift karta hai. Jaise: H₂O (2 neeche dikhta hai).
10 sup – Superscript
Text ko upar shift karta hai. Jaise: 2² (square ka 2 upar hota hai).
11 code – Code Tag
Programming code ya command ko show karne ke liye use hota hai.
12 span – Span Tag
Inline container hota hai, jisko CSS se style kiya ja sakta hai. Koi default style nahi hota.
13 label – Label Tag
Form fields ke liye text ya naam assign karne ke liye use hota hai.
14 input – Input Tag
User se input lene ke liye (text, number, checkbox, etc.).
